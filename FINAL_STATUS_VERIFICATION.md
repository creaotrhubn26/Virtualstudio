## 🎓 Developer Guide

### To Use Measurement Tool
1. Press 'M' or click Measure button
2. Click first point on canvas
3. Click second point to create line
4. Measurement auto-saves to shot

### To Use Status Workflow
1. Click status icon next to shot
2. Select new status from dropdown
3. Status updates immediately
4. Auto-saves to database

### To Export Scene
1. Click More menu (three dots)
2. Choose export format:
   - JSON (complete data)
   - PDF (professional report)
   - CSV (production data)
3. File downloads automatically

### To Add Floor Plan
1. Click "Upload Floor Plan" button
2. Select PNG or JPG image
3. Adjust scale with slider
4. Floor plan displays as background
